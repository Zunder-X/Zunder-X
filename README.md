## Featured project

### 🔭 Galaxy Morphology Classification
Automatic classification of galaxies into ten morphological categories using the Galaxy10 DECaLS dataset (17,736 images). Compares a custom CNN, ResNet50 and DenseNet121 with progressive fine-tuning, dropout and attention mechanisms (SE and CBAM).

**Best result: 86% accuracy · Macro F1 0.85**

[![Galaxy Repo](https://img.shields.io/badge/View%20repository-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/Zunder-X/Galaxy-Morphology-Classification)
